# Estimate and Invoice Generator

## Setup Instructions

### Environment Variables
To run the application, you need to set the following environment variables:
- `DATABASE_URL`: Connection string for the database.
- `API_KEY`: Your API key for external services.

You can create a `.env` file in the root directory of the project and add the variables like this:
```
DATABASE_URL='your_database_url'
API_KEY='your_api_key'
```

### Dependencies
Make sure to install the necessary dependencies. Run the following command in your terminal:
```
npm install
```

### Running the App Locally
To run the application locally, use the following command:
```
npm start
```
Once the server is up, you can access the application at `http://localhost:3000`.
